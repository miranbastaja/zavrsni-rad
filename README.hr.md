# Završni rad
Ovaj repozitorij sadrži typst kod za moj srednjoškolski završni rad. Tema rada
je to-do web aplikacija u [rust](https://www.rust-lang.org/) programskom jeziku
sa [actix-web](https://actix.rs/) frameworkom.
